Client's Journey — Step by Step
How a client goes from receiving a link → to submitting an inquiry for their dream plot
Phase 1 — Entry & Access
1
Client Receives the Platform Link
🔗
AUGUST sends a personalised link to the client via WhatsApp, SMS, or Email. The client taps the link on their phone or tablet — no app download needed. It opens instantly in the browser.
WhatsAppSMSEmailMobile & Tablet
Link Opens
2
Enter Mobile Number → Request OTP
📱
Client sees a clean login screen. They enter their mobile number and tap "Get OTP". The system sends a request to the AUGUST admin to approve the access. No password needed — just a one-time code.
System Sends OTP RequestAwaiting Admin Approval
Admin Reviews Request
⚖️ Admin Decision — Approve or Reject Client Access?
✅ Approved → Client receives OTP on their phone, enters it, and gains full access to the platform
❌ Rejected → Client is notified. Access is blocked. AUGUST team can follow up directly.
Access Granted
Phase 2 — Exploration
3
See Raipur City Map — 3 Township Locations Pinned
🗺️
Client lands on an interactive city map of Raipur. Three colourful pins mark the township locations. Tapping any pin shows a quick preview card with the project name, photo, and a "Explore" button. The AI chatbot is available in the corner to answer any questions.
📍 Water Lilly — Kachna📍 Silver Stone — Labhandi📍 White Lotus — VIP Road🤖 AI Chatbot Available
Client Taps a Project Pin
4
Zoom into the Township — See All Plot Cutouts
🏙️
The map smoothly zooms into the selected township. Client sees the full geo-coded layout with roads, zones, and every individual plot drawn to scale. Each plot is colour-coded so the client instantly knows what's available.
🟢 Available Plots 🟡 Reserved 🔴 Booked / Sold
Client Taps a Plot
Phase 3 — Plot Discovery
5
View Detailed Plot Information
📐
A detail panel slides up showing everything about the selected plot — visually presented with icons and clean layout. The client doesn't need to ask any questions. All the key info they'd care about is right there.
📏 Plot Size (sq.ft / sq.yards) 🧭 Facing Direction 🪷 Vastu Details & Score 🌳 Nearby Amenities 🏘️ Neighbour Plot Availability 💰 Price (if admin enabled)
Client Wants to Visualise
Phase 4 — 3D House Visualisation
6
Pick a House Design & See it on the Plot
🏠
Client browses a gallery of 5–10 pre-loaded house designs. They tap one — and instantly see a 3D model of that house sitting right on their selected plot. They can rotate, zoom in, and switch between day and night lighting. If they love how it looks, they can share a screenshot via WhatsApp.
5–10 House Designs Modern / Traditional / Contemporary Rotate & Zoom 🌙 Day / Night Toggle Share Screenshot
Client is Interested — Ready to Enquire
Phase 5 — Inquiry Submission
📩 Client chooses how they want to connect with AUGUST:
📞 Request a Call — "Call me at my preferred time"
💰 Get a Quote — "Send me a price breakdown"
🏗️ Book Site Visit — "I want to visit in person"
Short Form Submitted
7
System Automatically Creates a Lead & Alerts Admin
⚡
The moment the client submits, the system does everything automatically — a new lead card appears in the AUGUST admin dashboard, tagged with the client's name, plot ID, project, inquiry type, and timestamp. Admin is notified instantly. Client receives a confirmation message.
Auto Lead Created in Dashboard Admin Notified Instantly Client Gets Confirmation
After Submission
Phase 6 — Feedback & Continued Use
⭐ Rate This Plot
Client gives 1–5 stars and selects feedback tags (Vastu, Size, Location, Price…). Optional short comment. All of this goes into the admin's feedback analytics — helping AUGUST understand what clients love and what they don't.
🤖 Ask the AI Chatbot
Available throughout the entire journey. Client can ask: "Which project is closer to my office?", "Which project has a park nearby?", "What's the largest plot in Silver Stone?", "Tell me about Vastu compliance here."
🔁 Keep Exploring
Client can go back to the map at any time, explore other plots or other projects, switch between townships, and submit multiple inquiries — all in one session without losing their place.
🎯
Client Journey Complete

🖥️
Admin opens the internal dashboard on any device — desktop, tablet, or phone. The home screen shows a live overview of everything happening: new leads today, active client sessions right now, and which project is getting the most attention this week.

Ongoing Admin Operations
Phase 4 — Ongoing Operations
📋 Update Plot Status
When a deal is confirmed, admin marks a plot as Reserved or Booked. This update reflects immediately on the client-facing map — no plot is ever shown incorrectly to another client.
👁️ Watch Active Sessions
Admin can see which clients are currently browsing the platform right now — what they're looking at, which plots they've clicked, and how long they've been exploring. Useful for timing follow-up calls.
➕ Add New Projects
When AUGUST launches a new township, admin adds it via the content panel — new map data, plot information, and 3D models. The platform updates immediately. No technical team needed.
